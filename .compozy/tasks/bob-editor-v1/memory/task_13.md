## Important Decisions

- **size-limit.json**: Added all pipeline deps (`unified`, `remark-*`, `rehype-*`, `mermaid`, `monaco-editor`) to `ignore` list. Rationale: tsup externalizes all `dependencies` by default, so these don't end up in the dist bundle; size-limit was re-bundling them from node_modules. With correct ignore list, core = 16 KB gzip (well under 80 KB), full = 19 KB gzip (well under 500 KB).
- **jest-dom setup**: Fixed by using explicit `expect.extend(jestDomMatchers)` instead of relying on side-effect import. TypeScript fix via `tests/jest-dom.d.ts` with `/// <reference path=...>`.
- **InsertTable a11y test**: Opened dialog directly via component props (no toolbar button exists for it).
- **Bench baseline**: Updated to actual machine values (47/235/490ms p50) since original baseline was set for a faster machine.

## Learnings

- tsup automatically externalizes all `dependencies` (not just `external` array) — dist/index.js is only 96 KB because it imports unified/remark/rehype as ESM imports rather than bundling them.
- size-limit `ignore` array must reflect the same external packages that tsup externalizes; otherwise size-limit re-bundles all node_modules and reports inflated sizes.
