# bob-editor

> React Markdown editor component — work in progress.

A fully-featured Markdown editor React component with Monaco editing engine, GFM, math (KaTeX), Mermaid diagrams, alerts, and a plugin system.

## Packages

| Package                               | Description                   |
| ------------------------------------- | ----------------------------- |
| [`bob-editor`](./packages/bob-editor) | Published NPM package         |
| [`playground`](./apps/playground)     | Local dev app (not published) |

## Development

```bash
pnpm install
pnpm --filter bob-editor dev    # library watch build
pnpm --filter playground dev    # playground HMR
```
